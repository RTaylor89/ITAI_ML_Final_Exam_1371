# Credit Risk Dataset Source URL

**Dataset name:** Credit Risk Dataset  
**Kaggle dataset identifier:** `laotse/credit-risk-dataset`  
**Kaggle dataset page:** `https://www.kaggle.com/datasets/laotse/credit-risk-dataset`  
**Kaggle API download URL used:** `https://www.kaggle.com/api/v1/datasets/download/laotse/credit-risk-dataset`

This project uses the credit risk dataset exactly as downloaded from Kaggle. The original raw CSV is kept unchanged, and all cleaning work is performed in Python inside the notebook.
